countdownCube

Example 1

      $('#counter').countdownCube( {
        target: '2018-03-01T10:00:00Z',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'blue',
        } );

Example 2

      $('#counter-days-only').countdownCube( {
        target: '2018-03-01T10:00:00Z',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'red',
        labelsTranslations: {'year': 'anni',
                             'month': 'mesi',
                             'day': 'giorni',
                             'hour': 'ore',
                             'minute': 'minuti',
                             'second': 'secondi'
                             },
        showDaysOnly: true,
        } );

Example 3

Timezone support can be added with the moment.js and moment-timezone.js libraries.

      $('#counter-timezone-default').countdownCube( {
        target: '2018-03-01T10:00:00',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'forestgreen',
        } );
      $('#counter-timezone-new-york').countdownCube( {
        target: '2018-03-01T10:00:00',
        targetTimezone: 'America/New_York',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'green',
        } );
      $('#counter-timezone-los-angeles').countdownCube( {
        target: '2018-03-01T10:00:00',
        targetTimezone: 'America/Los_Angeles',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'chartreuse',
        } );

Example 4

If the built-in Date is used then the local time of the browser is inferred and the timezone argument is effectively ignored, beware that this behavior could be inconsistent across browsers.

      $('#counter-date').countdownCube( {
        target: new Date( 'March 1, 2018 10:00:00' ),
        targetTimezone: 'America/Los_Angeles',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'brown',
        } );
    
      $('#counter-date-utc').countdownCube( {
        target: new Date( '2018-03-01T10:00:00Z' ),
        cubeSize: 150,
        background:  '#ffffff',
        color: 'saddlebrown',
        } );

Example 5

      in65seconds = new Date();
      in65seconds.setSeconds(in65seconds.getSeconds() + 65);
      $('#counter-end').countdownCube( {
        target: in65seconds.toISOString(),
        targetTimezone: 'UTC',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'grey',
        onEnd: function(e) {
          $("#counter-end").text('This is the end!');
        }
      } );

Example 6

      $('#counter-end-notrigger').countdownCube( {
        target: '2017-12-28T10:00:00',
        targetTimezone: 'UTC',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'darkcyan',
        onEnd: function(e) {
          $("#counter-end-notrigger").text('This was the end, but it was not triggered!');
        }
      } );
    

      $('#counter-end-trigger').countdownCube( {
        target: '2017-12-28T10:00:00',
        targetTimezone: 'UTC',
        cubeSize: 150,
        background:  '#ffffff',
        color: 'cyan',
        onEnd: function(e) {
          $("#counter-end-trigger").text('This was the end and it was triggered!');
        },
        triggerEnd: true,
      } );
